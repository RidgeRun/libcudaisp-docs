Detailed Description

This class is in charge of the Debayer algorithm using CUDA.

It performs Debayering and it is constructed when passing kDebayer to the CudaBackend

Inherits from IAlgorithm. The conversion is from a 1 channel 8-bit unsigned packed CFA grayscale Bayer pattern to a 3 channel 8-bit unsigned packed RGB conversion. It uses the NPP library from NVIDIA.
