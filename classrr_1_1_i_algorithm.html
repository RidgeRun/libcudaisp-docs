Parameters
input_buffer: Input buffer for which the algorith will be applied.
output_buffer: Output buffer for which the algorithm has already been applied.

async: Indicates if the execution of the algorithm will be asynchronus or synchronus. If false, blocks the execution until algorithm finishes its execution. Default value: false.
