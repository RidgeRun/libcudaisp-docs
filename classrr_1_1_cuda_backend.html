Detailed Description

This interface class manages the creation of CUDA Algorithms and CUDA Buffers.

The CudaBackend::CreateAlgorithm and CudaBackend::CreateBuffer methods build Cuda Algorithms (Look at the Algorithms enumeration to see available options) and CudaBuffers respectively.
