Detailed Description

This class is in charge of the CudaWhiteBalancer algorithm.

Inherits from IAlgorithm. This class can recieve an input RGB image, it then applies automatic white balancing to the image, and then produces an RGB output. It uses the NPP library from NVIDIA to provide CUDA acceleration to the algorithm.
